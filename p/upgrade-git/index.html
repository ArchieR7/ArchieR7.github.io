upgrade git

Jun 01, 2018

前言

git 最近被發現有重大的漏洞，以我們能做的事情就是更新自己的 git！

在 macOS 上更新

在 macOS 上，預設的 git 為 git version 2.15.1 (Apple Git-101)， 所以我們需要先透過 brew 來安裝 git。 首先，我們先來更新 brew，這點和 CocoaPods 類似，需要更新本機端的項目； 順道升級 brew 目前裡頭的項目。

brew update && brew upgrade

接著就透過 brew 來安裝 git

brew install git

最後再將 Symbolic link 處理一下就好了！

brew link --force git

現在我們確認一下 git 是否已經為新版本（git version 2.17.1）！

git --version
